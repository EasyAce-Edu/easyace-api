# Structure
- Environment: NodeJS v0.12.x
- Main Framework: ExpressJS
- DB: MongoDB v3.x.x
- Linter: JSHint/ESLint (*ESLint is preferred*)

# Run Locally
```javascript
  node app.js or nodemon app.js
```
